Project Name: RDNT

Project Description:

RDNT is a project that takes an input audio source and runs signal processing algorithms to communicate with LED devices to create a synchronized audio-visual experience. Our device will create communication channels between LED devices and the microcontroller and from an input audio source to external speakers. Signal processing of an input audio source will be done by the microcontroller, which controls the LED devices based on preset behavior. The preset behaviors can be changed by the end user through a connected application that will feature external controls that will be used to select color ranges and maintain connections to LED devices.
