Project Name: RDNT

Project Description:

RDNT is a project that takes an input audio source and runs signal processing algorithms to communicate with wired LED devices to create a synchronized audio-visual experience. Our device will create communication channels between LED devices and the microcontroller and from an input audio source to external speakers. Signal processing of an input audio source will be done by the microcontroller, which controls the LED devices based on preset behavior. The preset behaviors can be changed by the end user through a connected application that will feature external controls that will be used to select color ranges and maintain connections to LED devices.

Project Specific Success Criteria (PSSCs):

  1. PSSC #1 (Software): The ability to control our LED devices via a mobile device using bluetooth.
  2. PSSC #2 (Hardware): The ability to amplify and filter analog input of the microphone using an operational amplifier.
  3. PSSC #3 (Hardware): The ability for the user to send and switch between two forms of input : microphone and aux.
  4. PSSC #4 (Hardware): The ability to regulate power down from 12V DC to 5V DC with a voltage regulator.
  5. PSSC #5 (Hardware): The ability to drive multiple WS2812Bs with PWM signal and configuring specific patterns in resposne to audio signals.
